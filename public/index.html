Random Excuse Generator API

Get hilarious excuses to bail out of any situation — or just to crack yourself up.

How to Use

  • GET /api/excuses — List all excuses
  • GET /api/excuses/random — Get one random excuse
  • GET /api/excuses/:id — Get excuse by ID
  • POST /api/excuses — Add a new excuse (send JSON like { "excuse": "Your excuse here" })
